## App Features: 

- Display News coming from the API.
- Search for News.
- Open the News Details in AFSafariViewController.
- Support Dark Mode.

## App Technologies:
 
* App Current Version: V1.0
* Supported IOS : IOS 13 or above
* Swift Frameworks : UIKit - SafariServices 
* Pattern : MVVM
* Supported languages in App : English
